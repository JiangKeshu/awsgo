awsgo
=====
awsgo is a command line tool used to list and connect to linux ec2 instances.

Requirements
============
- ``python >= 2.7``(awscli require version >= 2.7)
- ``ruby >= 1.8``
- ``awscli >= 1.1.0``http://aws.amazon.com/cli

Installation
============
1. Install python
2. Install awscli
> pip install awscli
